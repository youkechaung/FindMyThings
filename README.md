# 找得到 (Find Things)

一个简单的物品管理应用，帮助你记录和追踪物品的位置。

## 功能特点

- 添加物品（名称、描述、位置）
- 支持拍照或从相册选择图片
- 物品使用状态管理
- 搜索功能（按名称、描述、位置）
- 简洁的用户界面

## 技术栈

- SwiftUI
- PhotosUI
- UserDefaults 数据持久化

## 系统要求

- iOS 15.0+
- Xcode 14.0+

## 安装说明

1. 克隆仓库
2. 使用 Xcode 打开项目
3. 运行项目

## 使用说明

1. 点击右上角"+"按钮添加新物品
2. 输入物品信息（名称、描述、位置）
3. 可以选择拍照或从相册添加图片
4. 在主界面可以搜索物品
5. 使用"使用"按钮标记物品状态
